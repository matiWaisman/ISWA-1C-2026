## Ejercicio 1 
| input | lineas cubiertas | energia      |
|-------|------------------|--------------|
| ""    | [2]              | $1/7^5 = 1/16807$ |
| "a"   | [2]              | $1/7^5 = 1/16807$ |
| "b"   | [2, 3]           | $1/3^5 = 1/243$   |
| "c"   | [2]              | $1/7^5 = 1/16807$ |
| "d"   | [2]              | $1/7^5 = 1/16807$ |
| "aa"  | [2]              | $1/7^5 = 1/16807$ |
| "ab"  | [2]              | $1/7^5 = 1/16807$ |
| "ac"  | [2]              | $1/7^5 = 1/16807$ |
| "ba"  | [2, 3, 4]        | $1/1^5 = 1$       |
| "bb"  | [2, 3]           | $1/3^5 = 1/243$   |
| "bc"  | [2, 3]           | $1/3^5 = 1/243$   |
| "bad" | [2, 3, 4, 5]     | $1/1^5 = 1$       |

Como tabla "intermedia": 

| camino       | frecuencia |
|--------------|------------|
| [2]          | 7          |
| [2, 3]       | 3          |
| [2, 3, 4]    | 1          |
| [2, 3, 4, 5] | 1          |

La probabilidad de que el fuzzer elija el input "bad" para mutarlo va a ser igual a: 1/(sumatoria de las energias) que es igual a 1/2,01 = 0,49.
